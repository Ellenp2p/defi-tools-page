Yeap 存款 / 可赎回金额计算器

目标：只依赖 Aptos Fullnode REST API，估算“我当前等价存了多少 USDC/USDT”和“现在能取出多少”。

结论（先看这个）

  1. 核心是份额模型：可赎回资产 = 用户份额 × 当前每份额资产价值。
  2. 若你有池子总资产和总份额：每份额价值 = 池子总资产 / 池子总份额。
  3. 因此：可赎回 = 用户份额 × 池子总资产 / 池子总份额。
  4. 若已知你初始投入本金：收益 = 可赎回 - 本金。

这与 Yeap 文档中的 Vault share 逻辑一致：份额数量通常不变，份额价值随收益变化。

方式 A：一键查询 preview redeem（推荐）

输入用户地址后，页面会自动查询 YEAP USDT / YEAP USDC 两种包装代币，并直接调用 preview_redeem 计算可领取数量。

方式 B：手动输入（兼容模式）

  • 当前可赎回（USDC/USDT）: -
  • 当前等价存款价值（USDC/USDT）: -
  • 当前收益（USDC/USDT）: -
  • 每 1 份额价值（USDC/USDT）: -

方式 C：从 Aptos Fullnode 读取（高级可选）

你只需要 Fullnode + 两个资源类型（用户仓位资源、Vault 资源），字段路径可自定义。

完整计划（你要做什么）

  1. 在 Yeap/Aptos 合约里确认：用户份额字段、Vault 总资产相关字段、Vault 总份额字段。
  2. 把字段路径填到“方式 B”，先读取数据，再自动填入“方式 A”。
  3. 点击“计算”，拿到可赎回金额、每份额价值、收益。
  4. 如果你要核对“当前 token 和实际 USDC/USDT 的关系”，就看“每 1 份额价值”。

提示：USDC/USDT 在 Aptos 常见 decimals = 6；份额 token decimals 取决于协议实现。
